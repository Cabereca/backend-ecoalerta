## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

First, run the development BD:

```bash
# start
docker-compose up
# stop
docker-compose down
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
